## 简介

平台提供多种 GPU 容器云计算规格供您选择，以满足不同虚拟化能力需求，支持广泛业务应用与服务场景。

各计算规格基于硬件能力差异分为多规格族，每个族采用不同 Intel 处理器、CPU/内存配比、GPU 显卡、云盘类型及网卡虚拟化方式，实现计算、存储与网络性能的差异化。规格族内部细分为多种实例规格，规格越高，性能越强。

## 说明：

- 在控制台页面可查看 GPU 实例价格及配置项（系统盘、数据盘、公网 IP 等）费用。该价格为参考价，以 PPIO GPU Cloud 控制台实际下单为准。
- 不同地域及可用区的实例规格有所不同，请以控制台实际展示为准。
- GPU 实例仅限在同一规格族内变更规格。详细信息请参考修改实例规格文档。
- 购买实例请参见购买 GPU 容器云计算型实例和购买高性能计算 GPU 型实例。
