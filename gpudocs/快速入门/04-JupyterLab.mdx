## Jupyter

Jupyter 是一个交互式笔记本界面，非常适合使用 Python 进行 AI/ML 开发。通过使用 Jupyter，您可以在浏览器中连接到界面，并打开任何可以下载为 .ipynb 文件的笔记本。

## 功能介绍

### Notebook

Jupyter Notebook 是一个开源的交互式计算环境，允许用户在浏览器中创建和共享包含实时代码、文本、数学方程、可视化和解释性文本的文档。它支持多种编程语言，如 Python、R、Julia、Scala 等，广泛用于数据科学、机器学习、教育和研究领域。您可以在 Notebook 中执行代码块，查看输出结果，并保存整个工作流程。

您可以在 JupyterLab 的工作区中新建交互环境。

### Terminal

Terminal 是集成在 JupyterLab 环境中的终端模拟器，允许用户直接访问命令行，执行系统命令和脚本。这意味着您可以在同一界面中进行数据分析、编写代码、运行 Jupyter Notebook，同时也能进行文件管理、软件安装等操作。

使用 JupyterLab 的 Terminal，您可以：

1. 执行操作系统命令，如文件操作、软件包管理等。
2. 运行 shell 脚本或批处理文件。
3. 与 JupyterLab 环境交互，例如安装 JupyterLab 扩展或内核。
4. 管理 JupyterLab 服务器，如启动、停止或重启服务。

要在 JupyterLab 中打开 Terminal，通常通过点击菜单栏中的 "File" -> "New Launcher"，然后在弹出的窗口中选择 "Terminal" 来创建一个新的终端。或者，您也可以通过快捷键（通常是 `Ctrl+Shift+T`）快速打开一个新的终端。
