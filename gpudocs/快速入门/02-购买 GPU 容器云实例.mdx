## 配置说明

| 配置项         | 说明                                                         |
| -------------- | ------------------------------------------------------------ |
| 云存储         | 允许您选择要附加到实例的云存储硬盘的类型。选项通常包括更快访问速度的 SSD（固态硬盘）以及更大容量但速度较慢的 HDD（机械硬盘），您可以在「存储管理」页面管理您的云存储 |
| CPU            | 您选择实例的 CPU 数量。CPU 数量将影响实例的处理能力和性能，更多的 CPU 对于计算密集型应用程序是有益的 |
| 内存           | 您指定每个实例的 RAM 大小，单位为 GB。内存对于应用程序的运行和实例的整体速度至关重要 |
| 支持 CUDA 版本 | CUDA 是 NVIDIA 开发的并行计算平台和 API 模型。如果您计划运行针对 NVIDIA GPU 优化的应用程序，如机器学习和 3D 渲染应用，选择 CUDA 版本很重要 |
| 端口           | 您可以根据您的服务需求暴露您的端口，平台将会为您的服务提供端口映射服务 |
| 环境变量       | 用于指定操作系统运行环境的参数，您可以直接在此设置环境变量，平台会自动为您初始化完成 |
| 计费方式       | 您可以选择按量付费或灵活计费的方式，支持按月、按周、按日     |
| 基础镜像       | 平台为您提供的镜像                                           |
| 私有镜像       | 您上传到平台镜像库的镜像，您可以在「我的镜像」中查看         |
| 自定义镜像     | 您指定的镜像名，平台默认会从 DockerHub 拉取，请确保镜像名输入正确 |
