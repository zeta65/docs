# 如何安装 GPU 驱动？

您可以根据实际情况选择直接创建已安装 GPU 驱动的实例，或在已有实例上自行安装 GPU 驱动：

## 创建已安装 GPU 驱动的实例

在创建 GPU 实例时，您可以直接使用已安装 GPU 驱动的实例：

1. **在创建实例页面的“镜像”中，选择公共镜像**：
   - 选择已预装 GPU 驱动的 GPU 版公共镜像。详情请参见 GPU 版镜像预装 GPU 驱动。

## 在已有实例上安装 GPU 驱动

若您在创建 GPU 实例时未选择自动安装 GPU 驱动，为确保正常使用 GPU 实例，请手动安装 GPU 驱动和 CUDA 工具包。参考安装 GPU 驱动和安装 CUDA 工具包进行操作。

## NVIDIA 驱动安装成功，但 `nvidia-smi` 命令无效？

**问题分析**：可能是 kernel-devel 和 kernel 版本不一致，导致驱动程序编译出错。

**解决方案**：在实例内运行 `rpm -qa | grep $(uname -r)` 命令查看 kernel 和 kernel-devel 的版本号，确保版本一致。若不一致，请从正规渠道下载对应的 kernel-devel 包，再重新安装驱动。

## 执行 `nvidia-smi` 命令查看的 CUDA 版本与实际安装版本不一致？

执行 `nvidia-smi` 命令查询到的 CUDA 版本代表 GPU 实例支持的最高 CUDA 版本，并不代表实际安装的 CUDA 版本。
